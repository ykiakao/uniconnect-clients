# UniConnect

Base inicial em Flutter para o aplicativo acadêmico UniConnect.

## Tecnologias

- Flutter
- Dart
- GoRouter
- Riverpod
- Material Design 3

## Como rodar

```bash
flutter pub get
flutter run
```

## Estrutura

```txt
lib/
  core/
    router/
    theme/
  features/
    auth/
    student/
    teacher/
    tasks/
    grades/
    chat/
  shared/
```
